# Ride-Sharing Demand & Revenue Intelligence

## Objective

Analyze NYC Yellow Taxi data to understand demand patterns, revenue behavior, and simulate a dynamic surge pricing strategy.

## Dataset

NYC TLC Yellow Taxi Trip Records (1 Month)

## Tools Used

- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL
- Power BI

## Project Structure

raw/ → Original dataset  
processed/ → Cleaned and transformed data  
notebooks/ → Analysis notebooks
